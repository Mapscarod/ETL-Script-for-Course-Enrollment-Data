# Google Sheets ETL for Course Enrollment Data

## Description
This project is a Google Apps Script that automates the integration, cleaning, and transformation of course enrollment data across multiple Google Sheets.

It consolidates participant information, calculates derived variables (such as age), and classifies records based on enrollment status.

## Features
- Data extraction from multiple sheets
- Participant-based data consolidation using ID
- Batch processing for performance optimization
- Automatic age calculation
- Record classification (approved / not approved)
- Structured data writing across multiple columns

## Technologies
- Google Apps Script (JavaScript)
- Google Sheets
